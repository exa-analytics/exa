#####################################
Installation
#####################################


Anaconda
#######################
Using anaconda or miniconda::

    conda install -c exaanalytics exa


Pypi
#######################
Using pip::

    pip install exa


Repository
#########################
Manually (or for a development installation)::

    git clone https://github.com/exa-analytics/exa
    cd exa
    pip install -e .
