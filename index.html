Exa¶

A framework for data science

This package provides a framework for programmatic file manipulation and data organization and processing. Editors allow creation of modular and efficient text parsing systems. Data objects extracted can be collected into Containers which facilitate analytics and visualization with the Jupyter notebook. See exa for more information.
