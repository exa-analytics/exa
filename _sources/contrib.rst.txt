Development
#############

Environment
-----------
For a development ready installation::

    git clone https://github.com/exa-analytics/exa.git
    cd exa
    pip install -e .

Git
----
We make the following suggestions when using git.

- Set line endings: ``git config --global core.autocrlf false``
